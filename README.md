# FreteFlow

Sistema Full Stack para gerenciamento de fretes e operações logísticas.

## Funcionalidades

- Cadastro de Transportadoras
- Cadastro de Motoristas
- Cadastro de Veículos
- Cadastro de Cargas
- Ordens de Transporte
- Rastreamento de Entregas
- Dashboard Operacional
- Autenticação JWT

## Tecnologias

### Backend
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL

### Frontend
- React
- TypeScript
- Material UI
